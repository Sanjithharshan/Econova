# EcoNova Quick Start Guide

## 🚀 Get Started in 30 Seconds

### Option 1: Direct Browser
1. Double-click `index.html` to open in your browser
2. That's it! The page should load immediately

### Option 2: Local Server (Recommended)
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```
Then visit: `http://localhost:8000`

## 🐛 Troubleshooting

**Images not loading?**
- The images use Unsplash CDN
- Check your internet connection
- Images should load automatically

**Styles not applying?**
- Make sure `styles.css` is in the same folder
- Check browser console for errors

**JavaScript not working?**
- Make sure `script.js` is in the same folder
- Check browser console for errors
- Ensure you're using a modern browser
