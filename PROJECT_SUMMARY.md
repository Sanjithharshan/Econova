# EcoNova Project Summary

## 📋 Project Overview

**EcoNova** is a comprehensive waste management platform designed to create a sustainable ecosystem connecting authorities, citizens, and technology to build cleaner, greener communities.

- **Primary**: `#10b981` (Emerald Green) - Sustainability

## 🚀 How to Use

### Quick Start
1. Open `index.html` in any modern browser
2. All styling and scripts are self-contained
3. No build process or dependencies required

### Testing Authentication
1. Click "Get Started" button in navigation
2. Toggle between Login/Register tabs
3. Fill out forms to test validation
4. Submit to see success animations

### Exploring Features
1. Scroll through all sections
2. Hover over feature cards
3. Watch statistics animate
4. Try the contact form
5. Test responsive design
